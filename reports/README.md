# Reports

## What is this?
Analysis, findings, and documentation from experiments and research projects.

## Why does it exist?
Experiments produce data; reports turn that data into knowledge. This directory holds written analysis that synthesizes experiment results into actionable insights.

## Report Types

- **Experiment Reports** — Detailed analysis of specific experiments
- **Comparison Reports** — Multiple approaches evaluated side by side
- **Literature Reviews** — Surveying existing research on a topic
- **Technical Deep Dives** — Explaining complex concepts thoroughly

## When to use?
- After completing significant experiments
- Before starting new projects (review past findings)
- Sharing knowledge with collaborators
- Building personal reference material
