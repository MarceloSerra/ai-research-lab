# Tools

## What is this?
Shared utilities, helper scripts, and common code used across experiments.

## Why does it exist?
Avoid repeating yourself. Common operations like data loading, preprocessing, evaluation metrics, and visualization should be reusable across all experiments.

## Categories

- `data/` — Data loading and preprocessing utilities
- `evaluation/` — Metrics and scoring functions
- `visualization/` — Plotting and chart generation helpers
- `training/` — Training loop utilities, callbacks
- `utils/` — General-purpose helper functions

## When to use?
- Before writing new code, check here first for existing utilities
- After discovering a reusable pattern, add it here
- Share tools across different discipline experiments
