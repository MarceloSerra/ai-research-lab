# Computer Vision

## What is it?
Computer vision enables machines to interpret and understand visual information — images, videos, and 3D scenes. It transforms pixels into meaningful representations.

## Why does it exist?
Visual data is everywhere but needs interpretation:
- Medical imaging requires diagnosis assistance
- Autonomous vehicles must see their environment
- Quality control detects defects automatically
- Security systems monitor video feeds
- Content moderation filters inappropriate images

## When should I use it?
- Processing or analyzing visual data
- Building vision-based applications
- Automating visual inspection tasks
- Extracting information from images

## When should I NOT use it?
- Text data → Use [NLP](../nlp/README.md)
- Tabular data → Use [Classical ML](../classical_ml/README.md)
- Audio/speech → Use specialized audio models
