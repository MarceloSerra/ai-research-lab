# Notebooks

## What is this?
Interactive Jupyter notebooks for exploration, learning, and prototyping.

## Why does it exist?
Notebooks let you experiment with code interactively — perfect for learning new concepts, visualizing data, and prototyping before committing to structured experiments.

## Conventions

- Name files descriptively: `01_linear_regression_intro.ipynb`
- Each notebook should be self-contained and runnable from top to bottom
- Include markdown cells explaining the "why" not just the code
- Tag notebooks with difficulty: Beginner, Intermediate, Advanced

## When to use?
- Learning a new concept for the first time
- Exploring dataset characteristics
- Quick prototyping before formal experiments
- Visualization and debugging
