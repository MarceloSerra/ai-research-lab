# Natural Language Processing (NLP)

## What is it?
NLP enables computers to understand, interpret, and generate human language. It bridges the gap between human communication and machine processing.

## Why does it exist?
Language is the primary interface for human-computer interaction:
- Search engines need to understand queries
- Customer service needs to process complaints
- Research requires analyzing vast text corpora
- Communication across languages demands translation

## When should I use it?
- Processing or analyzing text data
- Building language-based applications
- Extracting information from unstructured documents
- Automating language-related tasks

## When should I NOT use it?
- Structured data already in database format → Use SQL/classical ML
- Non-text data (images, audio) → Use appropriate modalities
- Simple string matching suffices for the task
