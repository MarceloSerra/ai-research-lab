# Large Language Models (LLM)

## What is it?
Large Language Models are neural networks trained on massive text corpora that can understand, generate, and manipulate human language. They power chatbots, assistants, code generators, and creative tools.

## Why does it exist?
Language models enable capabilities previously impossible:
- Natural conversation with machines
- Code generation from descriptions
- Content creation at scale
- Complex reasoning over text
- Multi-step task automation through language

## When should I use it?
- Natural language understanding or generation needed
- Complex text processing beyond simple patterns
- Conversational interfaces
- Creative content generation
- Code assistance and automation

## When should I NOT use it?
- Simple pattern matching → Use regex/classical NLP
- Deterministic calculations → Use code directly
- High-stakes decisions requiring human judgment
- Tasks where hallucination is unacceptable without safeguards
