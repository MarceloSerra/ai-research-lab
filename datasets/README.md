# Datasets

## What is this?
A curated collection of datasets for ML experiments across all disciplines.

## Why does it exist?
Data is the foundation of machine learning. This directory organizes datasets by domain so you can quickly find what you need for each experiment type.

## Organization

- `classical/` — Tabular data for classical ML (CSV, Excel)
- `nlp/` — Text corpora for NLP tasks
- `vision/` — Image datasets for computer vision
- `embeddings/` — Pre-computed embeddings and vector databases
- `llm/` — Prompt/response pairs for LLM fine-tuning

## When to use?
When starting a new experiment, check here first before downloading external data.
